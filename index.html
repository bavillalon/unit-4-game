Crystal Collector Game!

Object of the game:

The object of the game is to add numbers associated with each of the crystals in to match a target number. The target number will be randomly generated at the beginning at the beginning of the game. The value associated with each crystal will also be generated randomly each game but will stay the same between each clicks within a game and will not be known to the player until it is clicked. Once the the player knows the values, he or she will need to figure out the best way to achieve the sum.
